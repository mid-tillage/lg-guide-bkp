## Description

## venv
Generate your virtual environment

```bash
cd 01_start
python -m venv env
```

Start it
```bash
cd 01_start
# Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass
.\env\Scripts\activate
```

## Installation

```bash
pip install -r requirements.txt
```

## Running the app
The following commands allow you to run the application.
